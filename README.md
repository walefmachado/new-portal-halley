# 🌌 Portal Halley — Educação Atuarial

O **Portal Halley** é uma plataforma digital dedicada à difusão de conhecimento, ferramentas pedagógicas e recursos interativos voltados para a área de **Ciências Atuariais**. O projeto foi idealizado e é desenvolvido em ambiente acadêmico, contando com a colaboração ativa de estudantes para criar um ecossistema completo de aprendizado, simulações e notícias do mercado securitário.

🌐 **Acesse o portal oficial:** [www.portalhalley.org](https://www.portalhalley.org)

---

## 🎯 Objetivos do Projeto

- **Apoio Didático:** Centralizar materiais de apoio, slides, notas de aula e exercícios práticos de programação (Python/R) voltados para atuarial e estatística.
- **Interfaces de Cálculo:** Disponibilizar aplicações interativas (como simuladores de juros simples/compostos e modelos de sobrevivência) para visualização e fixação de conceitos conceituais.
- **Inovação no Mercado:** Trazer notícias atualizadas, discussões sobre provisões técnicas, análise de risco e a distribuição de seguros (como o seguro rural no Brasil).

---

## 🛠️ Tecnologias Utilizadas

O ecossistema do Portal Halley utiliza uma combinação de tecnologias modernas para desenvolvimento web e computação científica:

- **Frontend:** HTML5, CSS3, JavaScript (estruturado de forma leve e estática, hospedado via GitHub Pages).
- **Computação Científica & Simulações:** - **Python:** Utilização intensiva de pacotes como `pandas` e `numpy` para análise estruturada de dados.
  - **R / Shiny:** Aplicações interativas para simulações de regimes financeiros e modelos atuariais.
- **Documentação & Textos:** LaTeX (Beamer para apresentações de slides e pacotes estatísticos para relatórios acadêmicos).
